Welcome to CSET

The Computer Software Engineering Technology program at Thaddeus Stevens College of Technology prepares students to design, develop, and build customized software programs. Students will learn basic programming, how to interpret specifications, application of software architecture, verification and validation principles, and software performance standards. The program emphasizes a practical hands-on education as software projects are required each semester.
